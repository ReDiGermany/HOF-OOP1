Kommando zum packen: javac Crashkurs.Uebung1/*.java && jar cfe StupidLogin.jar Crashkurs.Uebung1.StupidLogin Crashkurs.Uebung1\*.class && java -jar StupidLogin.jar
Kommandos zum benutzen:
	java -jar StupidLogin.jar
		Gibt die GUI variante zurück
	java -jar StupidLogin.jar [benutzername] [passwort]
		Gibt das Resultat in der shell zurück
